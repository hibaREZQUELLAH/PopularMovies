# Movies App

__Développement d'une application Android qui se charge de récupérer des informations d’une API pour afficher une liste de films.__ 

__Voici les librairies utilisées:__

  Retrofit2 pour les appels réseaux
  
  Glide pour l’affichage des url d’image
